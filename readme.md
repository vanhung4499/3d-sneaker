# 3D SNEAKER

- A small project using BabylonJS to create 3D view
- The idea is from Nike By You

## Preview

- [3d-sneakers](https://3d-sneaker.vercel.app/#)

## What I use?

- HTML + CSS
- Bootstap
- Javascript ES6
- Browserify to build a bundle.js from ES6 JS code
- BabylonJS for 3d view

## Local run

- Clone this repos
- npm install
- npm run build
- Open **index.html**
